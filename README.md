# ComfyUI Exit Button

A simple ComfyUI extension that adds a draggable **Exit Button** to the application.  

![Exit Button](exit.png)


## Features

- Exits the ComfyUI application window.  
- Closes the Python CMD console automatically.  
- Fully **draggable** for easy placement.  

## Installation

1. Open the ComfyUI `custom-nodes` folder.  
2. Open a terminal (Command Prompt or PowerShell) in that folder.  
3. Run the following command to clone the repository:

git clone https://github.com/kreonxv/ComfyUI-ExitButton.git
